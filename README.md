ALBUMCLASH
\n AlbumClash is an album review and music news application! It is simple, allowing users to make an account, leave reviews, and check out some news without all of the fuss!

1. The file musicApp.sql is a dump of my database. Once you export this file into MySQL Workbench, it should create the DB, tables, and data.

2. Update backend/src/main/resources/application.properties with your DB credentials.

3. The documents folder includes images of all the diagrams: ERD, Sequence, Architecture, and Wireframes
